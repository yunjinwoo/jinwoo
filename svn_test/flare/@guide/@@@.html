구분 방법 효과
gZip gZip으로 문서 압축 도큐먼트 파일은 모두 gZip을 거치도록 설정. 80%감소효과있음.
CDN CDN사용 요청을 분산시켜 과도한 트래픽 줄임.
Expires or a cache control Header header에 기간 설정 1. IIS5.0, IIS6.0에 비슷한 이슈가 있음. -> Filetimestamp:ChangeNumber
2. 서버의 expires header 설정을 통해, 클라이언트의 브라우저에 캐싱되는 도큐먼트나 이미지들이 많아 불필요한 트래픽을 감소시킬 수 있음.
Configure Etags 이미지에 Etag부착 서버상의 컴포넌트와 브라우저의 캐쉬에 있는 컴포넌트가 서로 일치하는지를 알기위한 메커니즘.상황에따라설정.
Flush the buffer early flush()는 부분적으로 준비된 html을 브라우저로 미리 보내어, 서버가 html이외의 영역과 다른 작업을 수행하는 동안에, 미리html 코드를 받아 사용자에게 빠르게 보여줄 수 있음. 상황에따라설정.
