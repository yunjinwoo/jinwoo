== Description ==

당신의 블로그에 해당 블로그와 전체 웹에 대한 검색 서비스를 추가하실 수 있습니다.

== Installation ==

[한국어]

인스톨 방법:


* 워드프레스의 관리자페이지에서 인스톨을 할 경우 3번부터 보십시오.


1. 플러그인을 다운로드하고 압축을 풉니다.

2. h-naver-ajax-search 디렉토리(폴더)를 wp-content/plugins/ 안에 넣습니다.

3. 여러분의 워드프레스 관리자에의 플러그인 페이지에서 H Naver AJAX Search를 활성화 시킵니다.

4. 먼저 네이버 오픈 API 키를 획득합니다. http://dev.naver.com/openapi/register

5. 활성화가 잘 되었을 경우 검색 결과가 잘나타날 것이며 플러그인의 우측 두 버튼이 검색 결과를 나타내주고, 지워줄 것입니다.

6. 모든 피드백은 hyunsik@hyunsik.me에서 받습니다.


[English]  

Installation Instructions:

1. Download the plugin and unzip it (didn't you already do this?).

2. Put the h-naver-ajax-search folder's files into your wp-content/plugins/ directory.

3. Go to the Plugins page in your WordPress Administration area and click 'Activate' next to H Naver AJAX Search.

4. Go to Options > H Naver AJAX Search to configure your settings and click 'save'.

a. You will need an Naver API key first. http://dev.naver.com/openapi/register

b. Next you will need to enter CLOSED or OPEN for the displayed search results.

5. You can give feed back for H Naver AJAX Search to hyunsik@hyunsik.me
